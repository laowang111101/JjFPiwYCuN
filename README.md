# 图书管理系统 java1012

## 项目概述

图书管理系统是基于 Spring Boot + Vue 的前后端分离系统。该系统旨在提供便捷的图书信息管理、用户管理等功能。

## 技术栈

- 后端：Java、Spring Boot、Spring MVC、MyBatis Plus、Redis
- 前端：Vue、Element UI
- 构建工具：Maven
- 数据库：MySQL
- 系统环境：JDK 1.8

## 功能模块

### 图书管理

- 图书信息管理
- 图书类型管理
- 借阅信息管理

### 其他管理

- 用户管理
- 密码更改

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- JDK 1.8 或更高版本
- Maven
- MySQL
- Redis

## 目录结构

```
-- backend (后端)
  |-- src
  |-- pom.xml
-- frontend (前端)
  |-- src
  |-- package.json
-- docs (文档)
-- database (数据库)
```

## 核心亮点

- 前后端分离，便于维护与扩展
- 集成 Redis，提升系统性能
- 使用 Spring Boot，简化项目配置与部署
- 采用 Element UI，界面简洁易用

## 部署步骤

1. 准备运行环境：安装 JDK 1.8、Maven、MySQL、Redis
2. 初始化数据库：导入数据库文件至 MySQL
3. 编译后端代码：使用 Maven 进行编译
4. 部署前端代码：将前端代码部署至 Web 服务器
5. 运行后端服务：执行编译后的后端代码
6. 访问前端页面：通过浏览器访问前端页面，开始使用系统

请注意，实际部署过程中可能需要根据具体环境进行适当调整。
